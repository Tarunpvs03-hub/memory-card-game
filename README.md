# Pokémon Memory Game

A fun memory game built with React that challenges players to remember which Pokémon they have already clicked.

The objective is simple: **click every Pokémon only once.** Clicking the same Pokémon twice resets your current score.

---

## Live Demo

## https://memory-game-ckax.onrender.com

## Game Rules

- Click a Pokémon card to earn a point.
- Every click shuffles the cards.
- Don't click the same Pokémon twice.
- Click all 12 unique Pokémon to win the game.

---

## Features

- Dynamic Pokémon cards fetched from the PokéAPI
- Cards shuffle after every click
- Live score tracking
- Best score tracking
- Loading and error states
- Responsive design for desktop and mobile
- Clean modern UI with smooth hover effects

### Clone the repository


